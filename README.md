# 🚀 E-Space Solutions Database Management System

A modern database management system developed for **E-Space Solutions** to manage Mars colonization operations efficiently.  
This project handles colonist registrations, pilot management, trip scheduling, housing allocation, and E-Jet management using a structured relational database system.

---

## 📌 Project Overview

The system was designed to improve data management, reduce redundancy, enhance security, and support efficient decision-making for space colonization operations.

The application includes multiple modules for managing:

- Colonists
- Dependents
- Pilots
- Trips
- Houses
- E-Jets
- Jobs
- Authentication & Login Systems
